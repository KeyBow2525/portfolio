プログラミング授業実践

中学校でのプログラミング学習における、オセロゲーム制作の実践記録です。

授業資料（アーカイブ）

  • arrow_forward1時間目：アルゴリズムの基本
  • arrow_forward2時間目：HTML/CSSの基礎
  • arrow_forward3時間目：JavaScriptの導入
  • arrow_forward4時間目：盤面の描画ロジック
  • arrow_forward5時間目：石を置く・裏返す処理
  • arrow_forward6時間目：勝敗判定の実装
  • arrow_forward7時間目：UIの調整とデバッグ

制作期間

2025.6 ─ 2025.10

担当

要件定義 / 授業構想 / コーディング

使用技術・環境

HTML / CSS / JavaScript / Google Colab

課題とコンセプト

  • ▸実践的なプログラミング学習ができる授業づくり
  • ▸スクラッチでのプログラミング学習・iPadでの制作活動の限界を超えること
  • ▸クラウド上での運用によるファイル保存に関するリスク軽減
  • ▸7時間構成で、各時間で学ぶべきポイントを設定すること

工夫した点・実績

段階を踏んだ授業構成とハードウェアに依存しない制作活動の実現

段階を踏みながら制作できるようコードを分けて作成し、その授業内にポイントとなるコードを押さえながら進められるよう工夫しました。また、学校ごとにハードウェアが異なるため、ブラウザ上で実行できるGoogle Colabを用いることで、環境に依存しない授業づくりを実現しました。授業構想時の学校で導入されていたのがiPadだったため、一部のデザインはiPadでの実行を想定しています。

生徒の制作意欲向上と専門外担当授業の負担軽減

ただ制作するだけでなく、ポイントを押さえて授業を組み立てて実施したため、HTMLやCSS、JavaScriptがどういう役割を果たしているのか、どのコードを書き替えることでアレンジができるかが理解できていました。これにより、完成後も意欲を持って授業に参加し、周囲の生徒と教え合う姿も多く見られました。また、技術科の教諭不足により専門外の教員が担当していた状況において、その負担軽減にもつながりました。
